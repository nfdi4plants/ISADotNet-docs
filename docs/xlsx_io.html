XLSX

last updated at 2023-02-14 ISA-XLSX

https://isa-specs.readthedocs.io/en/latest/isatab.html

ISADotNet contains a fromString and toString function for the three ISA-XLSX files. These files follow the object defined by the official ISA-Json schema:

Investigation

https://isa-specs.readthedocs.io/en/latest/isatab.html#investigation-file

#r "nuget: ISADotNet.XLSX" open ISADotNet.XLSX let investigationFilePath = "" let investigation = Investigation.fromFile investigationFilePath Investigation.toFile investigationFilePath investigation Assay/Study

https://isa-specs.readthedocs.io/en/latest/isatab.html#study-and-assay-files

open ISADotNet.XLSX.StudyFile let studyFilePath = "" let study = Study.fromFile investigationFilePath Study.toFile studyFilePath study open ISADotNet.XLSX.AssayFile let assayFilePath = "" let contacts,assay = Assay.fromFile investigationFilePath Assay.toFile assayFilePath contacts assay
